# 项目介绍

- 爱组搭 panshi 磐石 ~ 稳定可靠的 Web 开发框架。

> 使用必须遵守国家法律法规，⛔不允许非法项目使用，后果自负❗

- 愿景：每个人都是架构师
- 出处：《荀子·劝学》“锲而不舍，金石可镂”
